## Descripción
Este es un proyecto de práctica para aprender **Node.js** y **Express**.

## Características
- ✅ API RESTful
- ✅ Base de datos MySQL
- ✅ Autenticación JWT
- ✅ Documentación con Swagger

## Instalación
```bash
npm install
npm start

**Primeros pasos**

- Aprendiendo como ejecutar un archivo .js con Node.js
- Aprendiendo como funciona ESModule y CommonJS.


** Segundos pasos **

- Aprendiendo sobre el os - operating system.
- Aprendiendo que información se puede obtener del sistema operativo a través del entorno de trabajo Node.js.

** Tercer paso **

- Aprendiendo como funciona el system file - sistema de archivo, para manipular desde nodejs los archivo directamente del sistema operativo - operative system.
